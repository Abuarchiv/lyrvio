# Lyrvio Beta-Tester Guide

Beta-Phase: 10 Friends-and-Family in Berlin/München für 30 Tage. Ziel: 5× mehr Besichtigungs-Einladungen messen.

## Was du als Beta-Tester bekommst

- 30 Tage kostenfreie Aktiv-Suche (regulär 79€/Mo)
- Direkter Zugriff zur Lyrvio-Macher (mich) bei Problemen
- Erfolgs-Bonus 50% reduziert (149,50€ statt 299€) wenn du eine Wohnung findest

## Was wir von dir brauchen

- Aktiv suchen (nicht nur Extension installieren und vergessen)
- Ehrliches Feedback zu Bewerbungs-Qualität
- Pipeline-Updates (gesehen / eingeladen / besichtigt / Vertrag) per Dashboard pflegen
- Bei Sperrung deines ImmoScout-Accounts sofort melden
- 30-Min-Call nach 2 und 4 Wochen für Feedback

## Setup (10 Min)

1. Email an beta@lyrvio.com mit:
   - Stadt + Wunsch-Bezirke
   - Mappe (Schufa + Gehaltsnachweis 3 Monate + Selbstauskunft) als PDF
   - Anschreiben das du bisher nutzt (zur Stil-Lernung)

2. Magic-Link kommt per Email → Account angelegt
3. Browser-Extension installieren (Chrome oder Firefox)
4. Profile final überprüfen im Dashboard
5. Suche aktivieren

## Erste 7 Tage

- Bot sendet im Schnitt 5-15 Bewerbungen pro Tag
- Antworten kommen direkt in dein normales Postfach
- Dashboard zeigt Pipeline live

## Wenn was schiefgeht

- ImmoScout zeigt CAPTCHA → Extension pausiert automatisch, kontaktiere mich
- Account-Sperrung → sofort melden, ich helfe beim Wiederherstellen
- Schlechte Bewerbungs-Qualität → Beispiel zurückschicken, wird sofort behoben

## Kommunikation

Telegram-Gruppe: t.me/lyrvio-beta (Einladungs-Link kommt per Email)

## Datenschutz

- Mappe + Profil: lokal in Extension (IndexedDB), niemand außer du hat Zugriff
- Server speichert nur: email, gesendet-count, response-rate
- Kündigung: alle Daten löschbar, Export per JSON
