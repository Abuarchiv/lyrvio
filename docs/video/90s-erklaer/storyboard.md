# Storyboard: 90s Erklär-Video (Landing-Page Hero)

**Format:** 16:9 · 1920×1080px · 90 Sekunden · 30fps
**Struktur:** 3 Akte — Problem (0–30s) · Lösung (30–65s) · Resultat (65–90s)

---

## AKT 1: DAS PROBLEM (0:00–0:30)

### Szene 1.1 — 0:00–0:05: Eröffnung
- **Visual:** Schwarzer Screen. Fade In: Eine deutsche Großstadt aus der Vogelperspektive (Berlin/München/Hamburg Stock-Footage oder Motion-Graphic). Wohnblöcke.
- **On-Screen-Text:** (keine Schrift — nur Bild)
- **Voiceover:** „Wohnungen finden in Deutschland. 2024."
- **Musik:** Ruhiger Ambient-Beat beginnt — leicht melancholisch
- **Cut:** Slow dissolve

### Szene 1.2 — 0:05–0:12: Zahlen schockieren
- **Visual:** Animierte Grafik: ImmobilienScout-Inserat erscheint. Counter unter dem Inserat springt hoch: „Anfragen: 0 → 47 → 213 → 600" — Echtzeit-Zähler in 5 Sekunden.
- **On-Screen-Text:**  
  Groß: `600 Bewerbungen`  
  Klein: `pro Berliner Inserat im Durchschnitt`
- **Voiceover:** „Ein einziges Inserat in Berlin bekommt im Durchschnitt über 600 Bewerbungen."
- **Musik-Cue:** Bass setzt ein
- **B-Roll-Hinweis:** Counter-Animation via Remotion

### Szene 1.3 — 0:12–0:22: Das manuelle Problem
- **Visual:** Screen-Recording: Jemand füllt manuell ein Bewerbungsformular aus. Kopieren, Einfügen, Tippfehler korrigieren. ImmobilienScout, dann ImmoWelt, dann eBay Kleinanzeigen — dreimal derselbe Prozess. Zeitstempel oben: „19:34 Uhr — 21:17 Uhr".
- **On-Screen-Text:** `2 Stunden. Ein Abend. Drei Portale.`
- **Voiceover:** „Wer sich ernsthaft bewirbt, verbringt Stunden. Auf jedem Portal neu ausfüllen. Foto anhängen. Anschreiben anpassen. Absenden."
- **[SCREEN-RECORDING MARKER: Mockup ImmobilienScout Formular-Ausfüllung]**
- **Musik-Cue:** Tension steigt

### Szene 1.4 — 0:22–0:30: Der Schmerz
- **Visual:** Die Wohnungssuche — Smartphone-Mockup. Push-Notification: „Dieses Objekt ist nicht mehr verfügbar." Dann: E-Mail-Postfach. Betreff: „Leider müssen wir Ihnen mitteilen..." × 5 Mails.
- **On-Screen-Text:** `Zu spät. Wieder.`  
  (roter Text, kurz eingeblendet)
- **Voiceover:** „Und wenn du fertig bist, ist die Wohnung schon vergeben. Oder du bekommst gar keine Antwort."
- **Musik-Cue:** Beat endet abrupt — kurze Stille

---

## AKT 2: DIE LÖSUNG (0:30–1:05)

### Szene 2.1 — 0:30–0:38: Lyrvio Intro
- **Visual:** Schwarzer Screen. Lyrvio-Logo erscheint — Lyra-Konstellation Starburst in Indigo. Logo-Text erscheint darunter. Clean, ruhig.
- **On-Screen-Text:** `Lyrvio`  
  `Dein automatischer Wohnungs-Bot für DACH`
- **Voiceover:** „Lyrvio übernimmt das. Du richtest dein Profil einmal ein — der Bot bewirbt sich für dich. 24 Stunden. Sieben Tage die Woche."
- **Musik-Cue:** Neues Thema — positiv, modern Synthie, aufgeräumt

### Szene 2.2 — 0:38–0:48: Extension Installation Demo
- **Visual:** Screen-Recording: Chrome Web Store. Lyrvio-Extension Installation. Kurz, 10 Sekunden.
- **On-Screen-Text:** `1. Extension installieren.`  
  `chrome://extensions — dauert 30 Sekunden`
- **Voiceover:** „Installation: 30 Sekunden. Die Chrome-Extension lädt sich direkt in deinen Browser."
- **[SCREEN-RECORDING MARKER: Chrome Web Store → Extension installieren → Icon in Toolbar erscheint]**

### Szene 2.3 — 0:48–1:00: Profil-Setup Demo
- **Visual:** Screen-Recording: Lyrvio-Extension öffnet sich. Profil-Formular: Name, Berufsbezeichnung, Einkommen, Haustiere, Selbstauskunft-Upload. Jedes Feld wird kurz ausgefüllt.
- **On-Screen-Text:** `2. Profil einmal pflegen.`
- **Voiceover:** „Dann füllst du dein Profil aus — einmal. Name, Beruf, Einkommen, deine persönliche Nachricht. Alles, was Vermieter sehen wollen."
- **[SCREEN-RECORDING MARKER: Lyrvio Extension UI — Profil-Felder]**

### Szene 2.4 — 1:00–1:05: Bot läuft
- **Visual:** Animierter Bot-Status: „Lyrvio ist aktiv" — grüner Punkt blinkt. Dann: Zeitraffer-Animation. Uhr dreht sich. Benachrichtigungen poppen auf: „Bewerbung gesendet — 3:14 Uhr · Bewerbung gesendet — 7:22 Uhr · Bewerbung gesendet — 11:03 Uhr."
- **On-Screen-Text:** `3. Bot läuft. Du lebst.`
- **Voiceover:** „Lyrvio sucht jetzt selbstständig. Sobald ein passendes Inserat erscheint — sendet der Bot deine Bewerbung. Als einer der Ersten."
- **Musik-Cue:** Upbeat, Energie steigt

---

## AKT 3: DAS RESULTAT (1:05–1:30)

### Szene 3.1 — 1:05–1:15: Ergebnisse
- **Visual:** Animiertes Dashboard: „Bewerbungen diese Woche: 47. Antwortrate: 31%. Besichtigungen vereinbart: 3." Balkendiagramm wächst.
- **On-Screen-Text:** `47 Bewerbungen. Eine Woche. Null Aufwand.`
- **Voiceover:** „In einer Woche — 47 Bewerbungen auf passende Inserate. Drei Besichtigungsanfragen. Ohne dass du einen Finger gerührt hast."

### Szene 3.2 — 1:15–1:22: Preismodell
- **Visual:** Saubere Preiskarte erscheint auf Indigo-Hintergrund. Zwei Komponenten klar getrennt.
- **On-Screen-Text:**  
  `79€ / Monat`  
  `+ 299€ Erfolgsbonus`  
  `(nur wenn du einziehst)`
- **Voiceover:** „79 Euro im Monat. Und 299 Euro Bonus — aber nur, wenn du tatsächlich eine Wohnung bekommst."

### Szene 3.3 — 1:22–1:30: CTA
- **Visual:** Lyrvio-Logo. Website-URL groß. Button-Animation: „Jetzt kostenlos testen" erscheint mit Hover-Effekt-Simulation.
- **On-Screen-Text:**  
  `lyrvio.com`  
  `Jetzt kostenlos testen →`
- **Voiceover:** „lyrvio.com — kostenlos ausprobieren. Wohnungssuche, wie sie sein sollte."
- **Musik-Cue:** Kurzer Resolution-Sting, Fade Out
