# Storyboard: 3min Beta-Tester Onboarding-Video

**Format:** 16:9 · 1920×1080px · 3 Minuten · 30fps
**Distribution:** E-Mail Welcome-Sequenz (eingebettet oder Loom-Link)
**Ziel:** Neuer Beta-Tester installiert Extension, richtet Profil ein, versteht erste Bewerbung

---

## Intro — 0:00–0:15

### Szene I.1 — 0:00–0:15
- **Visual:** Lyrvio-Logo kurz eingeblendet (3 Sekunden). Dann: Talking Head (Gründer, entspannt)
- **On-Screen-Text:** `Willkommen bei Lyrvio Beta`  
  (dezent, obere linke Ecke, 3s)
- **Voiceover:** „Danke, dass du dabei bist. In den nächsten drei Minuten bist du startklar. Ich zeig dir alles was du brauchst."
- **Ton:** Warm, persönlich — nicht corporate. Beta-Tester sind frühe Unterstützer.
- **Musik:** Kurzer Jingle 3s, dann unter Sprache: -25dB

---

## Schritt 1: Installation — 0:15–0:55

### Szene 1.1 — 0:15–0:30: Chrome Web Store
- **Visual:** Screen-Recording: Chrome öffnen → URL-Bar: `chrome.google.com/webstore` → Suche: „Lyrvio"
- **On-Screen-Text (Overlay):** `Schritt 1: Extension installieren`
- **Voiceover:** „Öffne den Chrome Web Store und suche nach Lyrvio. Klick auf Hinzufügen."
- **Cursor-Highlight:** Auf „Zu Chrome hinzufügen"-Button
- **[SCREEN-RECORDING MARKER: Chrome Web Store, Lyrvio-Seite, Installieren]**

### Szene 1.2 — 0:30–0:45: Permissions-Dialog
- **Visual:** Permission-Dialog erscheint. Cursor zeigt auf „Erweiterung hinzufügen"
- **On-Screen-Text:** `Berechtigungen erlauben — notwendig für automatisches Ausfüllen`
- **Voiceover:** „Du siehst einen Berechtigungs-Dialog. Lyrvio braucht Zugriff auf ImmobilienScout und ImmoWelt — damit der Bot auf diesen Seiten arbeiten kann. Klick auf Hinzufügen."
- **[SCREEN-RECORDING MARKER: Permission-Dialog → bestätigen]**

### Szene 1.3 — 0:45–0:55: Extension in Toolbar
- **Visual:** Lyrvio-Icon erscheint in der Chrome-Toolbar. Cursor klickt darauf.
- **On-Screen-Text:** `Fertig. Extension ist aktiv.`  
  (grüner Haken-Emoji weggelassen — einfach Text)
- **Voiceover:** „Das war's — Lyrvio ist installiert. Das Icon oben rechts zeigt dir den Status."
- **[SCREEN-RECORDING MARKER: Extension-Icon in Toolbar klicken]**

---

## Schritt 2: Profil einrichten — 0:55–1:45

### Szene 2.1 — 0:55–1:05: Profil-Tab öffnen
- **Visual:** Lyrvio-Extension öffnet sich. Profil-Tab ist aktiv.
- **On-Screen-Text (Overlay):** `Schritt 2: Profil einrichten`
- **Voiceover:** „Jetzt richtest du dein Profil ein. Das machst du einmal — danach musst du es nicht mehr anfassen."
- **[SCREEN-RECORDING MARKER: Extension-UI, Profil-Tab]**

### Szene 2.2 — 1:05–1:25: Felder ausfüllen
- **Visual:** Screen-Recording: Jedes Feld wird kurz gezeigt — Cursor stoppt kurz dabei
- **Felder (mit Voiceover-Erklärung pro Feld):**
  - Name: „Vollständiger Name — genau wie im Personalausweis"
  - Beruf: „Deine aktuelle Berufsbezeichnung"
  - Einkommen: „Netto-Monatseinkommen — Vermieter fragen das immer"
  - Haustiere: „Ja/Nein — ehrlich angeben"
  - Persönliche Nachricht: „Das ist dein Anschreiben. 2–3 Sätze reichen."
- **On-Screen-Text:** Pro Feld kurz der Feldname eingeblendet
- **[SCREEN-RECORDING MARKER: Alle Felder mit Cursor-Callout]**

### Szene 2.3 — 1:25–1:35: Dokumente hochladen
- **Visual:** Upload-Bereich. Cursor zeigt auf „Selbstauskunft hochladen" und „SCHUFA hochladen"
- **Voiceover:** „Optional: Du kannst deine Selbstauskunft und SCHUFA hochladen. Lyrvio hängt sie automatisch an jede Bewerbung an — das hebt dich von der Masse ab."
- **On-Screen-Text:** `Optional aber empfohlen`
- **[SCREEN-RECORDING MARKER: Upload-Bereich]**

### Szene 2.4 — 1:35–1:45: Speichern
- **Visual:** Cursor klickt „Speichern". Grüne Bestätigung erscheint.
- **Voiceover:** „Speichern — fertig. Dein Profil ist gesichert."
- **On-Screen-Text:** `Profil gespeichert.`

---

## Schritt 3: Suche konfigurieren — 1:45–2:15

### Szene 3.1 — 1:45–2:00: Suchfilter-Tab
- **Visual:** Auf „Suche"-Tab wechseln. Filter-Einstellungen erscheinen.
- **On-Screen-Text (Overlay):** `Schritt 3: Deine Suche einstellen`
- **Voiceover:** „Jetzt sagst du Lyrvio, wonach du suchst. Wähle deine Stadt, Preisrange und Größe."
- **[SCREEN-RECORDING MARKER: Suchfilter-Tab, alle Felder]**

### Szene 3.2 — 2:00–2:15: Demo-Konfiguration
- **Visual:** Live ausfüllen: Stadt → Berlin (Dropdown), Max. Miete → 1200€, Min. Größe → 40m², Stadtteile → Mitte, Prenzlauer Berg, Friedrichshain
- **Voiceover:** „Ich wähle Berlin, bis 1200 Euro, mindestens 40 Quadratmeter, und ein paar Stadtteile. Du kannst das jederzeit anpassen."
- **[SCREEN-RECORDING MARKER: Filter live ausfüllen]**

---

## Schritt 4: Bot starten — 2:15–2:45

### Szene 4.1 — 2:15–2:25: Bot aktivieren
- **Visual:** Großer „Bot starten"-Button. Cursor klickt darauf. Status wechselt auf „Aktiv" (grüner Punkt).
- **On-Screen-Text:** `Bot ist aktiv.`
- **Voiceover:** „Drück auf Start. Lyrvio beginnt sofort zu suchen."
- **[SCREEN-RECORDING MARKER: Start-Button → Aktiv-Status]**

### Szene 4.2 — 2:25–2:45: Was jetzt passiert
- **Visual:** Animierter Status-Feed in der Extension: „Scanne ImmobilienScout... · Kein neues Inserat · Scanne ImmoWelt... · Neues Inserat gefunden · Bewerbung gesendet auf [Inserat]"
- **On-Screen-Text (Overlay, nacheinander):**  
  `Bot scannt alle 5 Minuten`  
  `Du bekommst eine E-Mail bei jeder Bewerbung`  
  `Bot läuft auch wenn Browser minimiert ist`
- **Voiceover:** „Lyrvio scannt jetzt alle 5 Minuten auf neuen Inserate. Du bekommst eine E-Mail wenn eine Bewerbung rausgeht. Der Bot läuft auch wenn du den Browser minimierst."
- **[SCREEN-RECORDING MARKER: Status-Feed Live]**
