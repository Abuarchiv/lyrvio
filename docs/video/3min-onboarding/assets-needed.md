# Assets Needed: 3min Onboarding-Video

## Screen-Recordings (kritisch — das ist fast das ganze Video)

- [ ] **Chrome Web Store → Lyrvio Extension installieren**
  - Pfad: chrome.google.com/webstore → Lyrvio suchen → Installieren → Dialog bestätigen
  - Dauer: 30 Sekunden
  - Cursor-Highlight: Auf allen Klick-Punkten

- [ ] **Lyrvio Extension öffnen + Profil ausfüllen**
  - Alle Felder ausfüllen: Name, Beruf, Einkommen, Haustiere, Nachricht
  - Demo-Daten nutzen (kein echter Datenschatz sichtbar)
  - Dauer: 45 Sekunden

- [ ] **Dokument-Upload**
  - Selbstauskunft-PDF hochladen (Dummy-PDF reicht)
  - Dauer: 10 Sekunden

- [ ] **Suchfilter konfigurieren**
  - Stadt, Preis, Größe, Stadtteile einstellen
  - Dauer: 20 Sekunden

- [ ] **Bot starten + Status-Feed**
  - Start-Button klicken, Status wechselt zu Aktiv
  - Status-Feed zeigt Aktivität
  - Dauer: 25 Sekunden

## Talking Head (optional — Gründer kurz einblenden)

- [ ] **Intro-Clip** (0:00–0:15): 15 Sekunden Talking Head
  - iPhone oder Webcam
  - Kein besonderes Setup notwendig — Beta ist informal
  
- [ ] **Outro-Clip** (2:45–3:00): 15 Sekunden Talking Head
  - Gleiche Einstellung wie Intro

## Logo & Brand

- [ ] **Lyrvio Logo** (SVG) — für Intro-Screen und Outro
- [ ] **Extension-Icon** (PNG 128×128) — für Toolbar-Closeup

## Annotations & Overlays

- [ ] **Step-Counter-Overlay**
  - „Schritt 1 von 4", „Schritt 2 von 4", etc.
  - Stil: Indigo Badge, unten links
  - Tool: Descript-Annotations oder Remotion

- [ ] **Cursor-Highlight**
  - Tool: Mouseposé (macOS) oder Screenium 3
  - Gelber Kreis um Cursor bei wichtigen Klicks

- [ ] **Text-Callouts für Felder**
  - Labels die auf Felder zeigen: „Name", „Beruf", etc.
  - Tool: Descript oder DaVinci Resolve Fusion

## Musik

- [ ] **Background-Musik:** Sehr dezent, kein Beat, -25dB
  - Pixabay: „gentle background ambient", „calm minimal music"
  - Länge: 3 Minuten oder loopbar

- [ ] **Intro-Jingle:** 3 Sekunden (gleich wie 30s-Hook — Konsistenz)

## E-Mail-Thumbnail

- [ ] **Video-Thumbnail für E-Mail** (600×338px)
  - Lyrvio-Logo + Text: „3-Min Setup: Dein erster Bot-Lauf"
  - Großer Play-Button-Overlay (weiß, transparent)
  - Tool: Canva oder Figma

## Loom-Hosting (empfohlen für E-Mail)

- [ ] Loom-Account einrichten (kostenlos)
- [ ] Video hochladen → Link generieren
- [ ] Thumbnail in E-Mail einbetten → Link auf Loom zeigen
