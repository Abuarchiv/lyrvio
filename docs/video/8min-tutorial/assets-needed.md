# Assets Needed: 8min Tutorial-Video

## Kamera / Aufnahme (Talking Head)

- [ ] **Talking-Head-Setup**
  - Option A: iPhone 15 Pro, Querformat, 1080p30fps — reicht vollständig
  - Option B: Logitech Brio 4K Webcam (falls Schreibtisch-Setup)
  - Licht: Fenster links oder einfaches Ring-Licht (~30€)
  - Hintergrund: Neutral — Wand, Bücherregal, oder leicht unscharf (Bokeh)

- [ ] **Mikrofon**
  - Rode Wireless GO II (klipsbar) oder DJI Mic 2
  - Alternative: AirPods Pro als Mikrofon (funktioniert für YouTube-Draft)
  - Ziel: sauberer Ton, kein Hall

## Screen-Recordings

- [ ] **Kapitel 1: ImmobilienScout24 Live-Suche**
  - Echte Website, echte Suche (Berlin, 1-Zimmer, bis 900€)
  - Timestamps der Inserate sichtbar lassen
  - Tool: macOS Screenshot App oder Screenium 3

- [ ] **Kapitel 1: Manuelle Bewerbung ausfüllen (mit Timer)**
  - IS24-Kontakt-Formular live ausfüllen
  - Timer-Overlay: OBS-Timer-Plugin oder Screenflick-Overlay
  - Dauer: ca. 4–5 Minuten (für Demo — kann auf 60s gekürzt werden)

- [ ] **Kapitel 2: Lyrvio Extension Interface**
  - Alle Tabs zeigen: Profil, Filter, Status, Log
  - Sauber, keine Test-Daten

- [ ] **Kapitel 3: Full Live-Demo**
  - Chrome Web Store → Installation → Profil → Filter → Bot-Start → Bewerbung
  - Backup-Screenshots wenn live kein Inserat erscheint

## Animationen / Slides

- [ ] **Agenda-Slide** (Keynote oder Figma)
  - 4 Punkte mit Timestamps
  - Lyrvio Brand-Farben

- [ ] **Architektur-Diagram** (Figma)
  - Flussdiagramm: Extension → Portal → Matching → Senden
  - Einfach, schwarz-weiß mit Indigo-Akzenten

- [ ] **Stoppuhr-Overlay**
  - OBS-Plugin: „StopWatch" oder per Script
  - Zeigt live Sekunden während Formular-Ausfüllung

- [ ] **Ergebnis-Statistik-Slide**
  - „312 Bewerbungen · 31 Antworten · 8 Besichtigungen · 1 Wohnung"
  - Animiert aufgebaut (Keynote Magic Move oder Remotion)

## B-Roll-Footage

- [ ] **Berlin Stadtaufnahmen** (Wohngebiet, Straße)
  - Pexels: „Berlin street", „Berlin apartment building"
  - 2–3 Clips, je 5–10 Sekunden

- [ ] **Schreibtisch-Setup-Aufnahme** (optional)
  - B-Roll vom Setup selbst — schafft Authentizität

## Thumbnail

- [ ] **Thumbnail-Design** (1280×720px)
  - Talking-Head links: Founder mit leichtem Ausdruck (kein übertriebenes Gesicht)
  - Text rechts: „Berlin Wohnung in 3 Wochen" (groß) + „mit Bot" (Indigo-Badge)
  - Tool: Canva oder Figma

## Editing

- [ ] **Jump-Cut-Bearbeitung** — alle „Ähs" und Pausen rausschneiden
  - Tool: Descript (automatische Transkription + Cut) oder DaVinci Resolve
  - Descript empfohlen: Text editieren = Video editieren

- [ ] **Cursor-Callout-Highlights** für Screen-Recordings
  - Tool: Mouseposé (macOS, ~4€) oder Screenium 3 (Annotation-Overlay)

## Musik

- [ ] **Intro-Jingle:** 5 Sekunden, positiv
- [ ] **Background-Musik:** Sehr leise, -25dB unter Sprache, minimalistisch
  - Pixabay-Suche: „lo-fi study background", „minimal background music"
  - Lizenz: CC0

## End Screen

- [ ] **YouTube End-Screen-Template** (20 Sekunden)
  - Subscribe-Button + Nächstes-Video-Feld (YouTube-Standard)
  - Statisch oder kurze Animation — Canva-Template reicht
